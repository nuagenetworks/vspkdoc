.. _nuforwardingclass:

nuforwardingclass
===========================================

.. class:: nuforwardingclass.NUForwardingClass(bambou.nurest_object.NUMetaRESTObject,):

Contains the Forwarding Class and its usage for load balancing.


Attributes
----------


- ``fec_enabled``: Indicates if FEC (Forward Error Correction) is enabled for the Service Class.

- ``load_balancing``: Indicates whether the Service Class is used to used for load balancing in the forwarding path.

- ``forwarding_class``: Class of service to be used.Service classes in order of priority are A, B, C, D, E, F, G, and H.
