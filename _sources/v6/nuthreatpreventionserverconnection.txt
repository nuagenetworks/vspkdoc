nuthreatpreventionserverconnection
===========================================

.. class:: nuthreatpreventionserverconnection.NUThreatPreventionServerConnection(bambou.nurest_object.NUMetaRESTObject,):

Represents connection between VSD instance and Threat Prevention Server


Attributes
----------


- ``fqdn``: Threat Prevention Server FQDN or IP address

- ``node_info``: Array of the embedded resource Threat Prevention Node Info for each Threat Prevention node.

- ``status``: VSD instance connection status with Threat Prevention Server
