.. _nunsportinfo:

nunsportinfo
===========================================

.. class:: nunsportinfo.NUNSPortInfo(bambou.nurest_object.NUMetaRESTObject,):

This API provides a list of monitoring ports and a list of configured ports for a NSG


Attributes
----------


- ``wireless_ports``: List of configured wireless ports for the gateway

- ``monitoring_ports``: List of monitoring ports for the gateway

- ``ports``: List of configured ports for the gateway

--------
