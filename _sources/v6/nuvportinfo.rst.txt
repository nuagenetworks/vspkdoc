nuvportinfo
===========================================

.. class:: nuvportinfo.NUVPortInfo(bambou.nurest_object.NUMetaRESTObject,):

Represents the vport status info at individual gateway for ethernet segment group


Attributes
----------


- ``v_port_operational_state``: VPort operational state at this Gateway

- ``gateway_id``: UUID of the Gateway

- ``gateway_name``: Name of the Gateway
